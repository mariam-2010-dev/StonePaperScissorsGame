# Stone-Paper-Scissors Game (C++)

## Description
This is a simple **Stone-Paper-Scissors** game developed in C++ using **Visual Studio**.  
The game allows the player to play multiple rounds against the computer and shows the winner of each round as well as the final game winner.

---

## How to Play
1. Open the `.sln` file in **Visual Studio**.
2. Build the project (`Build > Build Solution`).
3. Run the project (`Debug > Start Without Debugging`).
4. Enter the number of rounds you want to play (1-10).
5. In each round, choose:
   - `[1]` Stone  
   - `[2]` Paper  
   - `[3]` Scissors
6. The program will display:
   - Your choice
   - Computer's choice
   - Round winner
7. After all rounds, the game displays the final result:
   - Total rounds
   - Player wins
   - Computer wins
   - Draws
   - Final winner
8. You can choose to play again after the game ends.
